pyquest
=======

pyquest: diffusion analysis of questionnaires.

This version of pyquest includes tri-geometric analysis.
Updated files:
* dual_affinity.py
* plot_utils.py
* questionnaire.py
* tree.py
and a new IPython notebook:
* JSTSP.ipynb
